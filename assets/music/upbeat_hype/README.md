# Upbeat Hype — Music Library

Add royalty-free upbeat/hype .mp3 files here for ranking/listicle format videos.

## What to Look For

Tracks in this category are used for the Ranking / Listicle format ("Top 5 Scariest..."). They should feel exciting and fast-paced to complement quick-cut visuals.

Good characteristics:
- Heavy bass drops and trap hi-hats
- Electronic, hip-hop, or hybrid orchestral beats
- Punchy and energetic from the first second
- Fast tempo: 120–145 BPM
- Length: 45–90 seconds or seamless loops
- Should feel like a countdown or reveal show intro

## File Requirements

- **Format:** MP3 (128 kbps minimum, 320 kbps preferred) or WAV
- **Filename convention:** e.g., `trap_countdown_01.mp3`, `hype_build_02.mp3`
- **No stems** — single mixed file only

## How to Add

1. Download files from the sources above
2. Drop the .mp3 files into this folder
3. Restart the worker: `docker compose restart worker`
