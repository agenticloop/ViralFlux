# Cinematic Epic — Music Library

Add royalty-free cinematic epic .mp3 files here for motivational format videos.

## What to Look For

Tracks in this category are used for Motivational / Stoic Quotes format videos. They should feel grand, inspiring, and emotionally powerful.

Good characteristics:
- Full orchestral arrangement: strings, brass, choir, percussion
- Strong rising structure — starts building, peaks at the climax
- Inspiring and empowering, not dark or tense
- Tempo: 80–110 BPM with dramatic hits
- Length: 60–180 seconds, or loops with a clear introduction
- Think: movie trailer, sports montage, "hero's journey"

## File Requirements

- **Format:** MP3 (128 kbps minimum, 320 kbps preferred) or WAV
- **Filename convention:** e.g., `epic_rise_01.mp3`, `orchestral_triumph_02.mp3`
- **No stems** — single mixed file only

## How to Add

1. Download files from the sources above
2. Drop the .mp3 files into this folder
3. Restart the worker: `docker compose restart worker`
